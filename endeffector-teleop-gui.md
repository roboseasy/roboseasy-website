# SO-ARM101 End-Effector Teleop - 사용자 매뉴얼

## 프로그램 소개

SO-ARM101 End-Effector Teleop은 SO-ARM101 로봇암의 말단 장치(End-Effector)를 키보드로 직접 제어하기 위한 GUI 애플리케이션입니다. P-제어와 2D 역기구학(IK)을 기반으로 동작하며, 실시간 카메라 피드와 관절 상태 모니터링을 지원합니다.

## 기능별 사용법

### 1. 연결 (Connection)

로봇과 통신하기 위해 먼저 시리얼 포트에 연결해야 합니다.

**사용 방법:**

1. USB 케이블을 컴퓨터에 연결합니다
2. **Port** 드롭다운에서 연결할 포트를 선택합니다 (`/dev/ttyACM*` 또는 `/dev/ttyUSB*`)
3. **⟳ Refresh** 버튼으로 포트 목록을 갱신할 수 있습니다
4. **Connect** 버튼을 클릭합니다
5. 연결 성공 시 상태 표시가 `Connected`로 변경되며 로봇이 초기 위치(Zero Position)로 이동합니다

**연결 해제:**

- **Stop** 버튼을 클릭하면 시작 위치로 복귀 후 연결이 해제됩니다

> ⚠️ **주의:** 연결 전에 캘리브레이션 파일이 `~/calibration/robots/so_follower/` 경로에 있어야 합니다.

---

### 2. 키보드 제어 (Keyboard Control)

연결 후 키보드로 로봇을 제어할 수 있습니다. **프로그램 창에 포커스**가 있어야 키 입력이 동작합니다.

| 키 | 동작 | 설명 |
|----|------|------|
| **Q** | Pan 감소 | shoulder_pan 감소 |
| **A** | Pan 증가 | shoulder_pan 증가 |
| **W** | EEF X 감소 | 말단 장치 X축 이동 (IK 자동 계산) |
| **S** | EEF X 증가 | 말단 장치 X축 이동 (IK 자동 계산) |
| **E** | EEF Y 감소 | 말단 장치 Y축 이동 (IK 자동 계산) |
| **D** | EEF Y 증가 | 말단 장치 Y축 이동 (IK 자동 계산) |
| **R** | Pitch 증가 | wrist_flex 자동 계산 반영 |
| **F** | Pitch 감소 | wrist_flex 자동 계산 반영 |
| **T** | Roll 감소 | wrist_roll 감소 |
| **G** | Roll 증가 | wrist_roll 증가 |
| **Y** | 그리퍼 열기 | gripper 열기 |
| **H** | 그리퍼 닫기 | gripper 닫기 |
| **X** | 정지 및 복귀 | 시작 위치로 복귀 후 연결 해제 |

> 키를 **누르고 있으면** 해당 방향으로 계속 이동합니다 (50Hz 제어 루프).

---

### 3. End-Effector 상태 모니터 (End-Effector State)

로봇의 현재 상태를 실시간으로 확인합니다.

| 항목 | 단위 | 설명 |
|------|------|------|
| EEF X | m | 말단 장치 X 좌표 |
| EEF Y | m | 말단 장치 Y 좌표 |
| Pan | ° | shoulder_pan 관절각 |
| Lift | ° | shoulder_lift 관절각 |
| Elbow | ° | elbow_flex 관절각 |
| W.Flex | ° | wrist_flex 관절각 |
| W.Roll | ° | wrist_roll 관절각 |
| Gripper | % | 그리퍼 개폐율 (0: 완전 닫힘, 100: 완전 열림) |
| Pitch | ° | 피치 오프셋 |

각 항목은 미니 바(Bar)와 수치로 동시에 표시됩니다.

---

### 4. 카메라 피드 (Camera Feeds)

연결된 카메라의 영상을 실시간으로 확인합니다.

**사용 방법:**

1. **⟳ Detect cameras** 버튼을 클릭하여 연결된 카메라를 탐색합니다
2. 탐지된 카메라마다 체크박스가 생성됩니다 (기본: 활성화)
3. 체크박스를 해제하면 해당 카메라 피드가 숨겨집니다
4. 카메라가 없으면 화면이 빈 상태로 표시됩니다

> 카메라 피드는 약 30fps로 자동 갱신됩니다.

---

### 5. Waypoints 모드

주요 위치를 슬롯에 저장하고, 저장된 위치로 즉시 이동하거나 순서대로 자동 실행할 수 있습니다.

**Waypoints 모드 전환:**

- 상단 바의 **📍 Waypoints 모드** 버튼을 클릭합니다
- **◀ Teleop 모드** 버튼을 클릭하면 키보드 제어 화면으로 돌아옵니다

#### 위치 저장

1. 키보드로 로봇을 원하는 위치로 이동합니다
2. **📍 Waypoints 모드** 버튼을 눌러 Waypoints 화면으로 전환합니다
3. **슬롯 번호** (1~20)를 선택합니다
4. **현재 위치 저장** 버튼을 클릭합니다

저장되는 정보: X, Y, Pan, Pitch, Roll, Gripper 개폐율

#### 저장된 포인트 목록

| 컬럼 | 설명 |
|------|------|
| 슬롯 | 저장 번호 (1~20) |
| X (m) | 말단 장치 X 좌표 |
| Y (m) | 말단 장치 Y 좌표 |
| Pan (°) | shoulder_pan 값 |
| Pitch (°) | 피치 오프셋 |
| Roll (°) | wrist_roll 값 |
| Gripper (%) | 그리퍼 개폐율 |
| 이동 | 해당 위치로 이동 |
| 삭제 | 해당 슬롯 삭제 |

#### 전체 순차 실행

- **▶ 전체 순차 실행** 버튼을 클릭하면 저장된 슬롯을 번호 순서대로 자동으로 이동합니다
- 각 포인트 간 이동 후 약 2초 대기 후 다음 포인트로 이동합니다
- 로봇 연결 상태에서만 동작합니다

#### Excel 내보내기 / 불러오기

Waypoints 데이터를 Excel 파일(`.xlsx`)로 저장하거나 불러올 수 있습니다.

- **📤 Excel 내보내기**: 현재 저장된 모든 Waypoints를 `.xlsx` 파일로 저장합니다
- **📥 Excel 불러오기**: 저장된 `.xlsx` 파일에서 Waypoints를 불러옵니다

Excel 파일 형식:

| 슬롯 | X (m) | Y (m) | Pan (°) | Pitch (°) | Roll (°) | Gripper (%) |
|------|-------|-------|---------|-----------|----------|-------------|
| 1 | 0.163 | 0.113 | 0.0 | 0.0 | 0.0 | 50.0 |
| … | … | … | … | … | … | … |

> **이동** 버튼을 클릭하면 로봇이 저장된 위치로 즉시 이동합니다. 로봇 연결 상태에서만 동작합니다.

---

### 6. 정지 및 종료

| 버튼 | 동작 |
|------|------|
| **Stop** | 시작 위치로 복귀 후 연결 해제 (연결 중일 때 활성화) |
| **Quit** | 프로그램 종료 |

## 기본 사용 흐름

```
1. 하드웨어 연결
   └ USB-시리얼 어댑터 + SO-ARM101 전원 연결

2. 포트 연결
   └ Port 선택 → Connect 버튼 → 초기 위치(Zero Position) 자동 이동

3. 키보드로 제어
   └ W/S → EEF X 이동 (IK 자동 계산)
   └ E/D → EEF Y 이동 (IK 자동 계산)
   └ Q/A → Pan 조절
   └ R/F → Pitch 조절
   └ T/G → Roll 조절
   └ Y/H → 그리퍼 열기/닫기

4. (선택) Waypoints 저장 및 재현
   └ Waypoints 모드 버튼 → 위치 저장 → 이동 버튼으로 재현
   └ 전체 순차 실행 → 슬롯 번호 순서대로 자동 이동
   └ Excel 내보내기/불러오기로 Waypoints 파일 관리

5. 종료
   └ Stop 버튼으로 시작 위치 복귀 후 종료
   └ 또는 키보드 X 키로 복귀
```

## 문제 해결

### 포트가 목록에 나타나지 않음

- USB-시리얼 어댑터가 제대로 연결되었는지 확인하세요
- **⟳ Refresh** 버튼을 클릭하세요
- 어댑터를 다른 USB 포트에 연결해 보세요
- Linux의 경우 `/dev/ttyACM*` 또는 `/dev/ttyUSB*` 포트가 있는지 확인하세요
- 시리얼 포트 접근 권한이 없을 수 있습니다. 아래 명령어로 `dialout` 그룹에 추가 후 재로그인하세요:
  ```bash
  sudo usermod -aG dialout $USER
  ```

### 연결 시 캘리브레이션 오류 발생

- `~/calibration/robots/so_follower/` 경로에 캘리브레이션 파일이 있는지 확인하세요
- lerobot의 캘리브레이션 과정을 먼저 완료하세요

### 키보드 입력이 동작하지 않음

- 프로그램 창을 마우스로 클릭하여 포커스를 확인하세요
- 버튼이나 드롭다운에 포커스가 있으면 키 입력이 차단될 수 있습니다

### 로봇이 움직이지 않음

- 연결 상태(`Connected`)를 확인하세요
- 로봇 전원 공급 상태를 확인하세요
- Stop 후 다시 Connect를 시도해 보세요

### 카메라가 감지되지 않음

- **⟳ Detect cameras** 버튼을 다시 클릭해 보세요
- `/dev/video*` 장치가 존재하는지 확인하세요

### Waypoints 이동 / 전체 순차 실행이 동작하지 않음

- 로봇이 연결된 상태에서만 이동 기능이 동작합니다
- Connect 후 다시 시도하세요

### Excel 내보내기/불러오기가 동작하지 않음

- `.xlsx` 형식의 파일만 지원합니다
