모터 제어 GUI

So-Arm의 Feetech 모터를 실시간으로 모니터링하고 직접 제어할 수 있는 그래픽 인터페이스입니다. 모터 불량 테스트, 위치/속도/토크 제어, 온도 모니터링 등 다양한 기능을 제공합니다.

AppImage 다운로드

주요 특징

단일 파일 실행 직관적인 인터페이스 실시간 모니터링 안전한 제어 모든 Linux 배포판 지원

주요 기능

모터 불량 테스트

각 모터의 동작 상태를 개별적으로 확인하여 고장 여부를 조기에 발견할 수 있습니다.

실시간 모니터링

현재 위치, 속도, 토크, 온도 등의 값을 실시간으로 확인할 수 있습니다.

직접 제어

텔레오퍼레이션 없이 슬라이더를 통해 모터를 직접 제어하여 동작 범위를 확인합니다.

디버깅 지원

이상 동작 발생 시 원인 파악을 위한 상세 정보를 확인할 수 있습니다.

시스템 준비

1
FUSE 설치

AppImage를 실행하려면 FUSE가 필요합니다.

sudo apt-get update && sudo apt-get install -y fuse libfuse2
2
USB 포트 권한 설정

모터를 USB로 연결하여 제어하려면 사용자를 dialout 그룹에 추가해야 합니다.

sudo chmod 666 /dev/ttyACM0
sudo chmod 666 /dev/ttyACM1
sudo usermod -aG dialout $USER
3
실행 권한 부여 및 실행

다운로드한 AppImage 파일에 실행 권한을 부여한 후 실행합니다.

chmod +x RobosEasyMotorControl.AppImage
./RobosEasyMotorControl.AppImage
재로그인 필수

USB 포트 권한 설정 후에는 로그아웃했다가 다시 로그인해야 권한이 적용됩니다.

GUI 사용 방법

모터 연결

  • Feetech 모터를 USB-TTL 어댑터로 PC에 연결
  • GUI 상단의 Port 드롭다운에서 USB 포트 선택 (예: /dev/ttyUSB0)
  • "Connect" 버튼을 클릭하여 모터와 연결

모터 제어

  • 모터 ID 설정: 제어할 모터의 ID 입력 (1-253)
  • 위치 제어: Position 슬라이더로 목표 위치 설정
  • 속도 제어: Speed 슬라이더로 회전 속도 조정
  • 토크 제어: Torque 슬라이더로 출력 토크 제한

모터 불량 테스트

개별 모터 테스트

  • 모터 ID를 1부터 순차적으로 입력
  • Position 슬라이더를 천천히 움직여 정상 반응 확인
  • 이상한 소음이나 진동 발생 여부 점검
  • 온도가 비정상적으로 높지 않은지 확인

불량 증상

  • 응답 없음: 명령을 보내도 모터가 전혀 움직이지 않음
  • 불규칙한 움직임: 모터가 떨리거나 갑자기 멈춤
  • 과도한 발열: 정상 작동 시에도 온도가 급격히 상승
  • 이상한 소음: 기어 마모나 베어링 손상으로 인한 소음
  • 위치 오차: 목표 위치와 실제 위치의 차이가 큼

불량 모터 발견 시

불량 모터를 발견했다면 즉시 사용을 중지하고 교체해야 합니다. 불량 모터를 계속 사용하면 다른 부품까지 손상될 수 있습니다.

문제 해결

FUSE 오류 발생

증상: dlopen(): error loading libfuse.so.2 / AppImages require FUSE to run.

해결: FUSE를 설치하세요.

sudo apt-get install -y fuse libfuse2
모터를 찾을 수 없음

증상: "No serial ports found" 또는 "Permission denied" 오류

해결:

sudo usermod -aG dialout $USER
# 로그아웃 후 다시 로그인

# 모터 연결 확인
ls -l /dev/ttyUSB* /dev/ttyACM*
프로그램이 실행되지 않음

방법 1: 추출 모드로 실행

./RobosEasyMotorControl.AppImage --appimage-extract
cd squashfs-root
./AppRun

방법 2: 실행 권한 확인

chmod +x RobosEasyMotorControl.AppImage

모니터링 시작으로 실시간 확인

1. 연결

모터와 통신하기 위해 먼저 시리얼 포트에 연결해야 합니다.

사용 방법

  • USB 케이블을 컴퓨터에 연결합니다
  • 포트 선택 드롭다운에서 연결할 포트를 선택합니다
  • 새로고침 버튼으로 포트 목록을 갱신할 수 있습니다
  • 연결 버튼을 클릭합니다
  • 연결 성공 시 LED 표시기가 초록색으로 변합니다

2. 모터 선택

연결된 모터를 검색하고 제어할 모터를 선택합니다.

사용 방법

  • 스캔 버튼을 클릭합니다
  • 프로그램이 ID 1~29 범위에서 모터를 자동으로 검색합니다
  • 진행률 창이 표시되며, 완료 시 발견된 모터 수가 표시됩니다
  • 모터 드롭다운에서 제어할 모터를 선택합니다
  • 핑 테스트 버튼으로 선택한 모터와의 통신을 확인할 수 있습니다

3. 모터 ID 설정

모터의 ID를 변경할 수 있습니다.

사용 방법

  • 변경할 모터를 선택합니다 (현재 ID가 표시됩니다)
  • 새 ID 입력란에 변경할 ID를 입력합니다 (0~253)
  • ID 변경 버튼을 클릭합니다
  • 확인 창에서 Yes를 선택합니다
  • 변경 완료 후 스캔을 다시 실행하여 목록을 갱신합니다

주의

ID 변경 후에는 반드시 모터를 다시 스캔해야 합니다.

4. 모터 제어

모터의 위치, 속도, 가속도를 조절하고 이동 명령을 내립니다.

위치
0 ~ 4095
속도
0 ~ 3400

0 ~ 254
사용 방법

  • 슬라이더를 드래그하거나 숫자를 직접 입력하여 값을 설정합니다
  • 이동 버튼을 클릭하면 설정된 값으로 모터가 이동합니다
  • 정지 버튼으로 이동 중인 모터를 즉시 정지시킵니다

토크 제어

  • 토크 ON: 모터에 토크를 활성화합니다 (모터가 힘을 유지)
  • 토크 OFF: 토크를 비활성화합니다 (모터가 자유롭게 회전)

모터가 움직이지 않음
  • 토크 ON 상태인지 확인하세요
  • 속도 값이 0이 아닌지 확인하세요
  • 모터 전원 공급 상태를 확인하세요
연결이 불안정함
  • USB 케이블을 교체해 보세요
  • 다른 USB 포트에 연결해 보세요
