실행 전 설정
1
FUSE 설치
AppImage 실행에 필요한 FUSE를 설치합니다.
sudo apt-get update && sudo apt-get install -y fuse libfuse2
2
USB 포트 권한 설정
사용자를 dialout 그룹에 추가합니다.
sudo chmod 666 /dev/ttyACM0 sudo chmod 666 /dev/ttyACM1 sudo usermod -aG dialout $USER
3
실행 권한 부여 및 실행
AppImage 파일에 실행 권한을 주고 실행합니다.
chmod +x RobosEasyMotorControl.AppImage ./RobosEasyMotorControl.AppImage
재로그인 필수

USB 포트 권한 설정 후 로그아웃 → 재로그인해야 권한이 적용됩니다.

  • USB 케이블을 교체해 보세요.
  • 다른 USB 포트에 연결해 보세요.
  • 시리얼 통신 케이블 접촉 불량을 확인하세요.

Keyboard Teleop GUI

SO-ARM 로봇 팔을 키보드로 직접 조작하는 데스크탑 GUI. 실시간 카메라 피드를 보면서 각 관절을 제어하고 관절 상태를 화면에서 바로 확인할 수 있습니다.

AppImage 다운로드
주요 특징
단일 파일 실행 실시간 카메라 피드 관절 상태 모니터링 비상 정지 기능 직관적인 키보드 제어
화면 구성
연결 (Connection)
포트 드롭다운에서 로봇의 시리얼 포트를 선택하고 Connect/Disconnect로 연결을 제어합니다.
카메라 피드
연결된 카메라를 자동 감지해 실시간 표시. 체크박스로 카메라별 on/off가 가능합니다.
관절 상태
6개 관절의 현재 위치를 바 차트와 수치로 실시간 확인합니다.
컨트롤 패드
키보드 레이아웃을 시각화. 현재 누른 키가 하이라이트됩니다.

2
연결
Connect 클릭 → 상태가 ● Connected로 바뀌고 포트 선택이 비활성화됩니다.

키보드로 로봇 관절을 제어합니다. Shift를 함께 누르면 이동 속도가 2배로 빨라집니다.

모니터링

정상: Disconnect → Quit / 비상: Space 키 → 즉시 연결 해제

AppImage가 실행되지 않는다

chmod +x를 실행했는지 확인합니다. FUSE가 없는 환경이라면 아래 명령을 사용하세요.

./keyboard-teleop-gui-x86_64.AppImage --appimage-extract-and-run
포트 목록이 비어 있다 (no ports found)

로봇이 USB로 연결되어 있는지 확인하고 Refresh를 누릅니다. 그래도 안 보이면 시리얼 포트 접근 권한 문제입니다. 아래 명령 실행 후 재로그인하세요.

sudo usermod -aG dialout $USER
Connect를 눌렀는데 연결에 실패한다

다른 프로그램(터미널, ROS 노드 등)이 같은 포트를 점유 중일 수 있습니다. 해당 프로세스를 종료한 뒤 다시 시도합니다.

카메라가 감지되지 않는다

카메라 USB 연결을 확인한 뒤 Camera Feeds 패널의 Refresh를 누릅니다. Cheese, OBS 등 다른 앱이 카메라를 점유 중이라면 먼저 종료합니다.

키보드 입력이 반응하지 않는다

GUI 창을 한 번 클릭해 포커스를 맞춥니다. 연결(Connect)을 완료하지 않은 상태에서는 키 입력이 로봇에 전달되지 않습니다.

End-Effector Teleop GUI

SO-ARM 로봇 팔의 엔드이펙터를 키보드로 직접 제어하는 GUI. P-제어와 2D 역기구학(IK)을 기반으로 동작하며, 실시간 카메라 피드와 관절 상태 모니터링을 지원합니다.

AppImage 다운로드
주요 특징
단일 파일 실행 실시간 카메라 피드 엔드이펙터 상태 모니터링 Waypoints 저장·재현 Excel 내보내기/불러오기
화면 구성
연결 (Connection)
포트 드롭다운에서 로봇의 시리얼 포트를 선택하고 Connect/Stop으로 연결을 제어합니다.
카메라 피드
연결된 카메라를 자동 감지해 실시간 표시. 체크박스로 카메라별 on/off가 가능합니다.
엔드이펙터 상태
EEF X/Y 좌표와 각 관절 각도를 미니 바와 수치로 실시간 확인합니다.
키보드 제어
키보드 단축키로 EEF XY 좌표를 이동하면 역기구학이 자동으로 관절각을 계산합니다.
기본 사용 흐름
1

2
포트 선택 및 연결
Port 드롭다운에서 포트를 선택하고 Connect 클릭 → 로봇이 초기 위치(Zero Position)로 이동합니다.

연결 필요

이동 버튼과 전체 순차 실행은 로봇이 Connected 상태에서만 동작합니다.

포트가 목록에 나타나지 않는다

USB-시리얼 어댑터 연결을 확인하고 ⟳ Refresh를 누릅니다. 그래도 안 보이면 시리얼 포트 권한 문제입니다.

sudo usermod -aG dialout $USER # 로그아웃 후 재로그인
연결 시 캘리브레이션 오류 발생

~/calibration/robots/so_follower/ 경로에 캘리브레이션 파일이 있는지 확인하세요. lerobot 캘리브레이션을 먼저 완료해야 합니다.

키보드 입력이 동작하지 않는다

프로그램 창을 마우스로 클릭해 포커스를 맞춥니다. 버튼이나 드롭다운에 포커스가 있으면 키 입력이 차단됩니다.

카메라가 감지되지 않는다

⟳ Detect cameras 버튼을 다시 클릭합니다. /dev/video* 장치가 존재하는지 확인하고, 다른 앱이 카메라를 점유 중이라면 먼저 종료합니다.

AppImage가 실행되지 않는다

chmod +x를 실행했는지 확인합니다. FUSE가 없는 환경이라면 아래 명령을 사용하세요.

./endeffector-teleop-gui-x86_64.AppImage --appimage-extract-and-run
